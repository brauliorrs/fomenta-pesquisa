# Fomenta Pesquisa

**Sistema automatizado para monitoramento de editais, bolsas e chamadas de pesquisa, com coleta em fontes públicas, fila editorial, geração de cards e publicação assistida.**

O **Fomenta Pesquisa** é um projeto em Python voltado à automação da descoberta, organização e divulgação de oportunidades acadêmicas e científicas. Ele monitora fontes públicas de fomento, consolida editais em arquivos versionados, extrai informações relevantes
